# ecosystem-simulator
Une simulation d'ecosysteme ou on a la vegetation,les herbivores et les carnivores
Ecosystème 2D

Description

Ce projet est une petite simulation d'un écosystème en deux dimensions. Il contient trois types d'entités: des carnivores, des herbivores et de la végétation. Les carnivores mangent les herbivores, et les herbivores mangent la végétation. Le but est de simuler un environnement simple avec des règles de survie.

Objectif

Ce projet a été réalisé dans un cadre d'apprentissage pour pratiquer la programmation, la gestion de L'ecosysteme, les déplacements d'entités, et les interactions entre elles.

Langage utilisé C++
